### 热锅上的项目🔥

- **市场警惕 IEO** 中心化交易所依托其本身的信任，开始进行了一场基于交易所的通证发行比赛。区块律动做了一个[全面的汇总](https://www.theblockbeats.com/ieo)，近阶段会有更多的项目选择IEO。但是，IEO在bitmax过山车的行情下，危险已经有所浮出。甜甜圈🍩友情提醒：**天下没有持续的免费午餐。**

- **市场热度** **USDT** BTC有效突破 $5000/枚，使得场外资金热度增加明显，直接影响了USDT OTC的价格，目前兑价破7，溢价到达4.5%,目前溢价依旧在近3%。这也侧面反映了市场开始从熊市中走出来了，目前远远没有进入牛市，不过市场已经确立走出熊市了，之后会是阴晴不定的震荡市。

- **IEO 平台币回落**： 平台币HT, BNB,Bibox在最近都有明显的回落，从上周说BIMX从 2200 聪上涨到 1万聪到今天又到了1800聪，做了一次5倍来回的过山车🎢之后。Bibox也在经历不小的回落，同时-  **Veriblock破发** 前几日在小B网上线的Veriblock项目已经破发，当时发行价为0.15美金，现在已跌至0.09美金。这意味着IEO找二级市场接盘的可能性正在逐步降低，**甜甜圈提示**所有参与IEO项目的小伙伴们注意风险。IEO可能快速坠落。

-  [GT 平台币IEO](<https://www.gate.io/article/16810>) 继众多平台后，gate也将发布自己平台币GT，并且采取当年HT的买点卡送平台币的方式进行。初次规则是抢购自己期望的档次的点卡，但是由于引起大量资金波动，在周末时紧急修改规则，变为每个账号最多每个套餐限购10份，按照申购份额，等比例分配GT的购买权，价格不变，仍为0.4USDT（首日）。GT总量为10亿，本次首发3亿，分不同阶段以不同方式发行，具体可以参考链接的规则。**关于gate token（GT）改规则后的变化**：规则更改后变化的结果就是流通盘的持币结构发生了变化，新规则下将出现钱越多，持币越多的情况，这里要注意三点：1.在预期牛市的情况下，有钱的大户有锁仓的动机，因此新规则下GT在开盘的换手绝对量会降低，庄家没有拉盘动机，但散户的买盘能拉升价格，需要分析谁是买盘的主力；2.专做二级的资本不一定会在一级买入很多币，因为如果要在一级买入大量币，意味着需要提前在gate交易所打入远大于未来买到手的币的提前量。出于安全和信任，亿级以上的资金会很谨慎，不一定有这种意向，将大量钱，如5亿，10亿直接打到交易所；3.Gate和赵东的renrenbit提供类似小贷的借钱业务，也可能出现和第二点完全不同的情况，大量散户用杠杆资金来申购。目前usdt的溢价很大可能就是因为此事。类似当初fcoin开放充值导致以太坊拥堵，目前usdt的转账也被拥堵，有需求的建议改为XRP和EOS。其余的分析可以参考[一篇文章](https://mp.weixin.qq.com/s/Tfz2TQvJ2noM_g3_7sssdg).

- **警惕模式币**：周末，“走模式”和“模式币”突然走红，TNB、SHE都出现了巨大的涨幅。模式本质上即为cx，通过城市节点、拉人头、多层次营销等方式来带动资金入场。由于IEO、比特币等赚钱效应令市场的风险偏好提升，导致此前一些低成本、低估值的小币种被资金看上，成为了纯资金盘，通过cx的方式，通过拉盘和巨大的涨幅成为了上周末市场的焦点。甜甜圈🍩友情提醒：**看看就好**

### 嗅觉: 

- [**USDT折溢价指数**](https://chainext.io/mood_detail/90001/USDT%20OTC%20Index)：Chainext发布过一款USDT折溢价指数，由于法币渠道受限，场外法币的折溢价能反映市场供需情况，间接体现资金流向。目前指数为102.73，自2018年9月以来最高值为102.43，已经超过了前高，反映场外资金正持续流入，若场内无法为高溢价提供持续的赚钱效应，则高溢价料无法维持。（指数的编制方法为取火币和coincola的场外数据和离案人民币汇率进行比较，会与快迅类溢价数据的有一定出入，但仍可以作为监控场外资金的一个参考）

- [**Deribit**](https://www.deribit.com/)：一家荷兰团队出品的数字货币期货和期权交易所，目前期货交易日交易额为2亿美金左右，期权日交易额1000万美金左右。虽然期货方面无法与bitmex等老牌期货交易所相提并论，但deribit提供了一个产品层面功能较为完善的期权交易所，期权的隐含波动率能一定程度反应市场的情绪，若未来该市场的期权流动性增长至一定规模，可以成为一个观测市场情绪的好地方。
