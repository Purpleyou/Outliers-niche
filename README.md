### Dapp未来的应用: 

- **应用** [Flux](https://flux.market/) Flux 由 Peter Mitchell、Jasper De Gooijer 和 Jascha Samadi 创立，是一个针对初创公司的 P2P 衍生品交易平台。在 Flux 上，用户可以通过预测初创企业的发展时间表来进行交易，这个时间表可能包括产品发布、投资回合和进展 (或挫折) 等。意味着你可以做空某只创业公司。虽然如何找到对手盘永远是此类预测平台需要面对的问题，但甜甜圈看到了开放式金融更多可能性
- Dipor:[Dipor](https://www.theblockcrypto.com/2019/04/11/introducing-dipor-libor-for-open-finance/) 是近期仿照libor提出的一个概念，随着makerdao、Compound、Dharma等借贷市场逐步成熟，未来在去中心化世界，也将需要一个类似libor的产品来综合反应当下的借贷协议的利率情况，以给与用户或运营者参考，短期来看，也能帮助当前最大的defi应用makerdao更好地对dai进行风控g管理。于是14名开放式金融的成员便于4月10日提出了该概念。
