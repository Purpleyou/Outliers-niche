### 热锅上的项目🔥

- **BTC 数字黄金还是被逼空**: BTC在周末上冲到了7500美金，远远超出很多人的预期，抢了几乎所有币种的风头，BTC的避险功能再次被不断提及和体现。而分布式和通证通也适时推出了BTC估值模型，类比黄金，看看BTC的估值应该到何处，[储备估值法估值BTC](https://mp.weixin.qq.com/s?__biz=MzUyMTkzNDA5OQ==&mid=2247486636&idx=1&sn=f8f826fc3f731438941ee57e3464e829&chksm=f9d2c8d0cea541c6de5924f2b0fc55d996c164ae8c9e788c2846cb05bbe9e6c9941d665fd2c5&mpshare=1&scene=1&srcid=0510bpZdfYhenjfy9wI2SHIu#rd)。10万一枚？小伙伴们可以仔细看看其中的逻辑，和**数字黄金**相比，另一个观点则是巨鲸介入逼空行为，Willy Woo特别介绍[逼空机制101](https://twitter.com/woonomic/status/1127051915903115264)。资金的博弈和标的的质量有很强的关系。所以甜甜圈认为，至少在巨鲸看来，BTC数字黄金还是被众人所认可的。

- [iExec RLC](https://iex.ec/): 重构去中心化云计算的项目，很老的一个项目，目前V3即将在2天后，5月15日发布。去中心化云计算是一个很重要的方向,目前在这个方向上的项目不多，因为门槛比较高的缘故。CTO 将会参加concensus 2019的大会进行[发言](https://twitter.com/geo_protocol/status/1126490233887965184)。这个项目的开发一直有序推进，向小伙伴们推荐！

- [Harmony](https://harmony.one/): Harmony项目是高性能底层公链，团队技术实力非常强，对于堆栈的每一层都想法，团队都是资深的工程师，并且背景涵盖了人工智能、机器学习、区块链、VR和编译领域。所以在传输网络(谷歌的UDP、5G移动网络)、共识协议(实用拜占庭容错、分片技术)和系统工具(unikerkernel、零拷贝流)都有独到的技术优势。这个项目5月15日上线binance launchpad。值得期待！

- [火币Prime将于5月22日上线第三期项目reserve protocol](https://reserve.org/) reserve是一个稳定币项目，稳定机制上前期采用的是100%抵押锚定usdt的路线，中期多资产抵押，未来希望逐步过渡到算法银行路线。拥有三种类型代币，reserve token是稳定币，RSR是促进稳定币稳定的权益代币，Collateral tokens 是支持reserve稳定的抵押资产代币。项目采用套利机制来保持稳定，当价格脱离1美元，采用正反套利的模式来使价格回归，较为简单。当价格下跌导致抵押资产不足时，将允许增发RSR代币来补充资金。项目亮点是豪华投资团队，包括coinbase、ngc,fenbushi，yc总裁、paypal联合创始人等大牌投资人。


- **项目预警** [ThunderCore](https://www.thundercore.com/)两次大额转账引起社区争议，最早中文社区停止运营。[该项目最早中文社区发文称ThunderCore团队的人品与格局皆有问题](https://mp.weixin.qq.com/s?__biz=MzUzOTgwNzc0NQ==&mid=2247483792&idx=1&sn=3af741dd0b0a8e354ba757550cd5dcd9&chksm=fac398fecdb411e829c33dce55dc8ea13d8c654ad633e3826b2d70210a8377cc81abb84801e6&mpshare=1&scene=1&srcid=05120yDX8HX7BYXPumCcK7bO&key=9ce4dc7bce72ad0baf193d717da123e66b48385c7e91c13b5dca863c67a9460241d0f72e016e290403856e72f144f7267169bcbb27ed7193485ecc2bcac7156dcd5b6f19d40482f2d52497484f904deb&ascene=1&uin=MTYyMjAwNTU0NA%3D%3D&devicetype=Windows+10&version=62060739&lang=zh_CN&pass_ticket=WeHRdmwg4oNVxefAVlxpr7aex5pOkvq9Uo7jrzZHGvJDyAGp0G5NhaEc7n8hiPCB)，近日停止运营。[团队回应](https://www.chainnews.com/news/068808179347.htm?from=timeline&isappinstalled=0)称转入Upbit和火币的两笔5000万及3000万TT币原计划将来作为员工持币发放，并没有抛售打算，为消除误解现已提回。
